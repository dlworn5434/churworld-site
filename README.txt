이 압축은 기존 Astro 프로젝트에 덮어쓰기용입니다.

추가/수정된 내용:
- 전체 디자인 업그레이드 (더 진한 프리미엄 다크톤, 강조선, 카드 입체감)
- 클릭 가능한 카드에 화살표/바로 보기 표시 추가
- 생활 콘텐츠 허브와 각 생활 상세 페이지 유지
- 상세 페이지 상단의 "문서" 라벨 제거 → 각 페이지 제목이 표시되도록 변경

적용 방법:
1. 압축 풀기
2. 기존 프로젝트의 src, public 폴더에 덮어쓰기
3. PowerShell에서 프로젝트 폴더로 이동
4. npm.cmd run dev
5. 문제 없으면 GitHub Desktop에서 커밋 후 Push
