# 츄르월드 서버 규칙

모든 유저가 즐겁게 플레이할 수 있도록 아래 규칙을 지켜줘.

## 1. 기본 예절
- 다른 유저를 비하하거나 욕설하지 않기
- 분쟁을 일부러 만들지 않기
- 운영진 안내에 협조하기

## 2. 플레이 규칙
- 남의 건축물이나 자원을 무단으로 훼손하지 않기
- 허가 없는 약탈, 트롤링, 괴롭힘 금지
- 서버 경제나 밸런스를 망치는 행위 금지

## 3. 금지 행동
- 핵, 오토, 매크로 등 불공정한 프로그램 사용 금지
- 버그 악용 금지
- 서버에 과부하를 주는 행위 금지

## 4. 채팅 규칙
- 도배 금지
- 과도한 분쟁 유도 금지
- 불쾌감을 주는 표현 금지

## 5. 처벌 안내
규칙 위반 시 경고, 일시 정지, 영구 차단 등의 조치가 있을 수 있어.

## 6. 마지막 안내
운영진은 서버 환경을 위해 필요 시 규칙을 수정할 수 있어.
최신 내용은 공지 페이지를 확인해줘.
