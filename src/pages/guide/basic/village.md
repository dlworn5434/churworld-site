# 마을이란?

마을은 특정 지역(청크)을 점유해서 우리만의 구역으로 만들고, 마을 레벨을 올리면서 인원, 땅, 경작지, 월드보더 같은 혜택을 늘려가는 콘텐츠야.

## 1. 마을 생성

### 생성 조건
- 한 유저는 마을 1개만 가입 가능
- 이미 다른 마을에 가입되어 있으면 마을 생성 불가
- 마을 생성 가능한 월드는 서버 설정에 따라 제한될 수 있어 (기본: `world`)

### 명령어
- `/마을 생성 <마을명>`

### 생성 보상
- 마을 생성 시 자동으로 **9청크(3×3)** 를 기본 점유해줘

## 2. 점유(클레임) 시스템

### 황무지란?
마을이 점유하지 않은 땅은 황무지로 취급돼. 황무지는 마을이 땅을 확장하면서 점유할 수 있어.

### 점유 방법
1. 점유하려는 청크로 이동
2. `/마을 땅 점유` 입력

### 점유 규칙
- 점유하려는 청크는 마을 땅과 인접해야 함
- 가로/세로 인접만 가능
- 대각선 인접은 불가능
- 점유는 **매니저 이상**만 가능
- 점유 시 기본 비용 **1000원**
- 마을 레벨이 높아질수록 비용이 증가할 수 있어

### 점유 해제
- `/마을 땅 해제`
- 해제해도 점유 비용은 환불되지 않음

### 점유 범위 확인
- `/마을 땅 범위`
- GUI에서도 확인 가능

## 3. 그룹(땅 관리 단위)

마을의 점유지는 그룹 단위로 관리돼.

### 그룹 특징
- 하나의 그룹이 여러 청크를 포함할 수 있어
- 하나의 청크는 한 그룹에만 속해
- 마을 생성 시 기본 그룹이 자동 생성
- 마을원이 가입하면 기본 그룹에 자동 추가

### 그룹이 중요한 이유
그룹별로 아래 항목을 따로 운영할 수 있어.
- 접근 가능한 멤버
- 권한(보호 / 사용 가능 기능)
- 경작지 지정 가능 여부

### 그룹 관리
- `/마을 땅 관리`

## 4. 마을원(멤버) / 초대

### 마을원 초대
- `/마을 초대 <닉네임>`
- 채널 / 월드 상관없이 초대 가능

### 수락하면?
- 마을에 가입됨
- 즉시 마을 월드 스폰으로 이동
- 기본 그룹에 자동 추가

## 5. 매니저

마을장은 마을원 중 매니저를 임명할 수 있어.

### 매니저가 할 수 있는 것
- 땅 점유 / 해제 관리
- 그룹 생성 / 이동 / 권한 설정
- 마을원 관리(일부 기능)
- 손님 초대 / 관리

### 관리 GUI
- `/마을 관리`
- `/마을 땅 관리`

## 6. 손님(외부인 임시 초대)

마을원이 아닌 외부인을 일정 시간 손님으로 초대할 수 있어.

### 손님 초대
- `/마을 손님 초대 <닉네임> [분]`
- 매니저 이상만 가능

### 손님 규칙
- 초대 수락 시 즉시 마을로 이동
- 만료 전까지 다른 손님 초대는 받을 수 없음
- 채널 이동 후 접속하면 자동 이동될 수 있음
- 손님이 나가기: `/마을 손님 퇴장`
- 강제 추방: `/마을 손님 추방`
- 권한은 방문자로 처리되며, 그룹 설정에 따라 제한될 수 있음

## 7. 경작지(커스텀 농사 구역)

마을의 점유지 중 일부를 경작지로 지정할 수 있어. 커스텀 농사 플러그인과 연동되는 구역이야.

### 경작지 특징
- 경작지는 그룹 기반으로 운영 가능
- 이미 경작지인 땅에는 중복 지정 불가
- 경작지 최대 개수는 마을 레벨에 따라 증가
- 기본은 1인당 1개 기준 운영 가능

### 확인 / 설정
- `/마을 땅 관리` GUI에서 경작지 확인 가능

## 8. 마을 레벨 & 조공

마을은 조공을 통해 경험치를 올리고 레벨업해.

### 레벨업하면 좋아지는 점
- 최대 인원 증가
- 점유 가능한 땅 증가
- 경작지 슬롯 증가
- 월드 보더 확장 등

### 조공 시스템
조공은 아이템을 바쳐 마을 경험치(XP)를 올리는 시스템이야.

#### 조공 GUI 열기
- `/마을 조공`
- 또는 `/마을 메뉴` → 조공 버튼

#### 조공 방법
1. 조공 GUI에 아이템을 넣어둠
2. `[조공하기]` 버튼 클릭

#### 검사 / 정리 버튼
`[검사/정리]` 버튼을 누르면:
- 조공 가능한 아이템만 남기고
- 조공 불가 아이템은 자동으로 반환됨
- 예상 XP 합계도 보여줌

#### 조공 목록 보기
조공 GUI의 `[조공 목록]` 버튼에서:
- 어떤 아이템이 조공 가능한지
- 개당 XP가 얼마인지
직접 확인 가능

## 9. 마을 메뉴 추천 사용법

보통 유저는 아래 흐름으로 사용하면 돼.

1. `/마을`로 마을 메뉴 열기
2. 땅 확인 / 범위 보기
3. 조공 넣고 레벨업 참여
4. 마을원이면 그룹 권한에 따라 상자 / 문 / 농사 등 이용

## 10. 자주 묻는 질문

### Q. 모루로 아이템 이름 바꾸면 조공 안 되나요?
아니야. 조공 아이템은 `Material / CMD(커스텀 모델 데이터)` 기준으로 인식돼서 이름을 바꿔도 동일하게 조공 처리돼.

### Q. 조공할 아이템이 뭔지 모르겠어요.
조공 GUI에서 `[조공 목록]` 버튼을 눌러 확인하면 돼. `[검사/정리]` 버튼을 누르면 조공 불가 아이템은 자동 반환돼.
