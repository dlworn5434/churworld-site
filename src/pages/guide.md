# 츄르월드 시작 가이드

츄르월드에 처음 온 걸 환영해!

## 1. 서버 접속
- 서버 주소: `chur.kr`
- 권장 버전: 추후 입력
- 서버에 접속한 뒤 기본 안내를 먼저 확인해줘.

## 2. 처음 해야 할 것
- 스폰 지역 안내 읽기
- 기본 규칙 확인하기
- 필요한 경우 운영진 안내 따르기

## 3. 추천 루트
1. 서버에 접속한다
2. 규칙을 읽는다
3. 기본 거점을 정한다
4. 다른 유저와 교류한다

## 4. 도움이 필요할 때
문제가 생기면 공지사항을 먼저 확인하고, 필요한 경우 운영진에게 문의해줘.

## 5. 꼭 기억할 것
츄르월드는 함께 오래 즐기는 서버를 목표로 해.
서로 배려하면서 즐겁게 플레이해줘.
