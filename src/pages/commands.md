# 명령어 모음

## 기본 명령어
- `/spawn` : 스폰으로 이동
- `/home` : 집으로 이동
- `/sethome` : 집 설정
- `/warp` : 워프로 이동

## 생활 / 편의 명령어
- `/mail` : 우편함 확인
- `/bag` : 가방 열기
- `/guide` : 길라잡이 열기
